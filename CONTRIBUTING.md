# Contributing to Elyan macOS Security

Thank you for your interest in contributing to Elyan macOS Security!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/your-username/elyan-macos-security.git`
3. **Create** a branch: `git checkout -b fix/your-fix`
4. **Make** your changes
5. **Test** your changes
6. **Submit** a Pull Request

## Security Considerations

- Never commit sensitive data or credentials
- Follow macOS security best practices
- Test on multiple macOS versions if possible

## Code Style

- Follow Swift coding conventions
- Use meaningful variable names
- Add comments for complex logic
